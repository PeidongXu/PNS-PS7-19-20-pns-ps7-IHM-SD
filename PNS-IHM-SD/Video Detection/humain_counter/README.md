# human_counter
基于yolo3的人数统计程序

---

# A program for counting the amount of persons in an image or a video

<br>1. 首先像yolo v3那样把model下载好，教程: https://github.com/qqwweee/keras-yolo3

<br>2. 主要运行people_flow.py(对图片)和flow.py(对视频)两个程序

```
python people_flow.py
```
